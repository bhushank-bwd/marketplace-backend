# marketplace-backend

## Initilaization

- Run `npm init -y`
- install packages

1. bcryptjs
2. cors
3. dotenv
4. express
5. express-validator
6. jsonwebtoken
7. mongodb
8. mongoose

## Add env variable

1. PORT=5000
2. DB_CONNECTION_STRING=mongodb://localhost:27017
3. DB_NAME=marketplace
4. JWT_SECRET=iamjwtkey

## Notes

1. use `ip addr show` to get ip address on linux
